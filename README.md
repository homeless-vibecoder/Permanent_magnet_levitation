This project set out to build a platform that hovers in mid-air using only permanent magnets: a fixed magnet array on the ground repelling magnets on a free-floating platform. I designed candidate layouts by hand, modeled the magnetic forces in Python (a magnetic surface-charge force model plus a gradient-descent optimizer that searched for the most stable layout), and built physical prototypes - a cardboard array of neodymium magnets and a 3D-printed triangular holder frame. The platform never levitated freely, and the project explains why: Earnshaw's theorem forbids stable static levitation with permanent magnets. This is still not a proof of impossibility as the theorem doesn't necessarily apply in all cases - perhaps there is some engineering one can do in order to break the assumptions made in the theorem. Future work is not out of question.
